Introducción

  • Las computadoras digitales usan números binarios para realizar sus actividades de procesamiento, cálculo, etc.
  • Por conveniencia, los humanos, utilizan un sistema decimal que es más comprensible para sus cálculos.
  • Debido a lo anterior es necesario conocer lo valores de un sistema a otro y las operaciones aritméticas en cada sistema.
  • A continuación se muestran algunas definiciones y se presentan los sistemas más importantes.

Sistemas posicionales.

Definición

Un sistema de numeración posicional se representa como: $$ N = \left(a_{n-1}a_{n-2}\ldots a_1 a_0 . a_{-1}a_{-2}\ldots a_{-m} \right)_r $$

Donde $r$ es la raíz, "$.$" es el punto de raíz $r$, $n$ es el número de dígitos enteros, $m$ es el número de dígitos fraccionarios, $a_{n-1}$ es el dígito más significativo (MSD) y $a_{-m}$ es el dígito menos significativo (LSD).

Sistemas posicionales.

Definición

Un sistema de numeración posicional se representa de forma polinomial como: $$ N = a_{n-1}\times r^{n-1} + a_{n-2}\times r^{n-2}+ \ldots + a_0 \times r^0\\ + a_{-1}\times r^{-1} + \ldots + a_{-m} \times r^{-m} $$

Por ejemplo: $$(51.41)_{10} = 5\times 10^{1} + 1\times 10^0 + 4\times 10^{-1}+ 1\times 10^{-1}$$

Sistema decimal

  • Nombre:
    • "Decem" (Latin) -> Diez .
  • Características:
    • Diez símbolos: -> $0, 1, 2, \ldots, 9$.
    • Posicional: -> $2945 \ne 2954$.
    • $2945 = 2\times10^3 + 9 \times 10^2 + 4\times 10^1 + 5\times 10^0$.

Sistema binario

  • Nombre:
    • "Binarius" (Latin) -> Dos .
  • Características:
    • Dos símbolos: -> $0, 1$.
    • Posicional: -> $1010 \ne 1100$.
    • $1010 = 1\times 2^3 + 0 \times 2^2 + 1\times 2^1 + 0 \times 2^0$.
  • Terminología:
    • Bit: -> Dígito binario.
    • Byte -> 8 bits.

Sistema octal

  • Nombre:
    • "Octo" (Latin) -> Ocho.
  • Características:
    • Ocho símbolos: -> $0, 1, 2, \ldots, 7$.
    • Posicional: -> $1743_o \ne 7314_o$.
    • $2645 = 2\times 8^3 + 6 \times 8^2 + 4\times 8^1 + 5\times 8^0$.

Sistema hexadecimal

  • Nombre:
    • "Hexa" (Griego) -> Seis.
    • "Decem" (Latin) -> Diez.
  • Características:
    • Dieciséis símbolos: -> $0, 1, 2, \ldots, 9, A, B, C, D, E, F$.
    • Posicional: -> $A13D_H \ne 3DA1_H$.
    • $2AE5 = 2\times 16^3 + 10 \times 16^2 + 14\times 16^1 + 5\times 16^0$.
